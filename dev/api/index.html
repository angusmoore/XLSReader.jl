Reading files

XLSReader.readxls — Function
readxls(path) -> Workbook

Read an XLS file and return a Workbook containing all worksheets. Only BIFF8 (Excel 97–2003) is fully supported; BIFF5 (Excel 95) may work for simple files. BIFF2/3/4 are not supported.

Example

wb = readxls("report.xls")
sheet = wb["Summary"]
value = sheet[2, 3].value   # row 2, column 3 (1-based)

XLSReader.readxlsheet — Function
readxlsheet(path, sheetindex::Int; skip=0) -> DataFrame
readxlsheet(path, sheetname::AbstractString; skip=0) -> DataFrame

Read a single worksheet from an XLS file and return it as a DataFrame. The first row (after skipping skip rows) is used as column names. Empty or blank header cells fall back to "Column$c". Data cells that are empty or blank become missing.

source
XLSReader.Sheet — Type
Sheet

A single worksheet within a Workbook.

Fields:

  • name::String — sheet name as it appears in the workbook
  • nrows::Int — number of rows containing data
  • ncols::Int — number of columns containing data
  • cells::Dict{Tuple{Int,Int},Cell} — sparse cell map keyed by (row, col) (1-based)

Use sheet[row, col] to retrieve a Cell. Positions outside the populated range return an empty cell (type == 0, value === nothing).

Example

wb = readxls("report.xls")
sheet = wb["Summary"]
for row in 1:sheet.nrows, col in 1:sheet.ncols
    cell = sheet[row, col]
    println(cell.value)
end
source
XLSReader.Cell — Type
Cell

A single cell value read from an XLS worksheet.

Fields:

  • type::Int — cell type:
    • 0 — empty (no data)
    • 1 — text; value is a String
    • 2 — number; value is a Float64
    • 3 — date/time; value is a Date or DateTime
    • 4 — boolean; value is a Bool
    • 5 — error (e.g. "#DIV/0!"); value is a String
    • 6 — blank (formatted but empty); value is nothing
  • value — the cell content typed according to type, or nothing for empty/blank cells

Example

wb = readxls("report.xls")
cell = wb["Sheet1"][2, 3]
cell.type == 2 && println(cell.value::Float64)

DataFrames.DataFrame — Method
DataFrames.DataFrame(sheet::Sheet) -> DataFrame

Convert a Sheet to a DataFrame. Each column of the sheet becomes a DataFrame column (named "1", "2", …). Empty and blank cells become missing.
